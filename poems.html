Format of these poems

Limerick: To make this poem, you need to use the AABBA rhyming scheme. The first two lines rhyme, and the 3rd and 4th lines rhyme and the last line rhymes with the first two. As you can see, this form of poetry has 5 lines. Couplet: This poem is very simple to make. It has two lines and each line rhymes. Haiku: This type of poem has three lines and the first and the last line have five syllables and the middle line has seven. Cinquain: This type of poem is a very different type. The first line has a vague description of an object in on word. The second line has two adjectives. The next line has two -ing ending verbs, and the next line has a phrase about it. The last line reveals the object.
